# 此项目实现类NavMesh寻路以及单位行为

目前已实现约束德劳内三角剖分，切耳法生成不规则地图，下一步是替换切耳法，并且优化代码逻辑，暂时对于所有查询未用任何加速结构，纯暴力遍历

## 使用手册

1. 运行后，逆时针点击以形成地图轮廓，右击以撤回。
2. 插入坐标完毕后点击右边的“Start Triangulation”以开启三角剖分[^1]。
3. 生成地图后可以点击生成建筑[^2][^3]，右击以撤回[^4]。
4. 点击“Clear”以清除所有。

[^1]: 目前使用切耳法生成地图
[^2]: 边长为5，四个角切掉1.5的建筑，使用整数坐标
[^3]: 使用约束Delaunay三角剖分生成
[^4]: 撤回使用Delaunay三角剖分

## 一些要点

1. 约束貌似并没有起到什么作用？
2. 如果刚生成一个建筑又撤回可能会因为切耳法切出的三角形不满足Delaunay三角剖分的空圆性质从而复原后的地图与原来的不同。
3. 三角网格的顶点目前采用点的索引实现，用于避免浮点数带来的误差导致查询出错。
