# 此项目旨在还原星际争霸2中的地图构建，寻路以及单位行为

目前已实现约束德劳内三角剖分生成不规则地图以及添加删除建筑，下一步将引入A*寻路并光滑其路径，暂时对于所有查询未用任何加速结构，纯暴力遍历

## 使用手册

1. 运行后，逆时针点击以形成地图外围轮廓[^1]，右击以撤回。
2. 插入坐标完毕后点击右边的“Start Triangulation”以开启三角剖分[^2]。
3. 生成地图后可以点击生成建筑[^3][^4]，右击以撤回[^5]。
4. 点击“Clear”以清除所有。

[^1]: 目前只支持单层结构
[^2]: 目前使用约束德劳内三角剖分生成地图
[^3]: 默认建筑为边长为5，四个角切掉1.5的八边形
[^4]: 使用约束Delaunay三角剖分生成
[^5]: 撤回使用Delaunay三角剖分
